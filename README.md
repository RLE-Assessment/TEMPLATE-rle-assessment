A template repository for building a Red List of Ecosystems assessment report.

## To clone the template

1. **Fork this template repository**

    - Go to https://github.com/RLE-Assessment/TEMPLATE-rle-assessment
    - Select **Use this template** > **Create a new repository**
        - Under **Include all branches** select **On**
        - Populate the **Repository name**. For example: "*ruritania-rle-assessment*"
        - Add a **Description** for the repository. For example: "*An IUCN Red List of Ecosystems assessment for Ruritania*"
        - Click **Create Repository**
2. **Create a local clone**
      - From the GitHub Web UI, select **<> Code** > **Local**
3. **Install the local environment**
    - If needed, [install the Pixi package manager](https://pixi.prefix.dev/latest/installation/) on your local computer.
    - Run `pixi install`
